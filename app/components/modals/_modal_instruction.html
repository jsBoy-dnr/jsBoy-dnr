Инструкция по регистрации на корпоративном портале

  1. Зайдите в Личный кабинет на корпоративном портале https://intranet.aimholding.ru/
  2. В разделе «Рабочие сервисы» на вкладке «Мои сервисы» выберите «Справки и документы» или «Расчетные листы», после чего будет выполнен автоматический переход на страницу регистрации пользователей в Личном кабинете
  3. Чтобы пройти регистрацию в Личном кабинете необходимо ввести номер мобильного телефона. (В дальнейшем на указанный номер телефона будут приходить уведомления от ЛК о готовности документов и других услугах).А также необходимо подтвердить своё согласие на обработку персональных данных и нажать кнопку «зарегистрироваться»
  4. В течение 30 сек. на указанный номер телефона придет смс с четырехзначным проверочным кодом, который необходимо ввести в строку «код подтверждения» и нажать кнопку «войти»
  5. После этого на указанный вами номер телефона с адреса SUEK будет направлено смс уведомление с «пин–кодом», который необходимо сохранить. «Пин-код» потребуется в дальнейшем для прохождения авторизации в случае работы с сервисами Личного кабинета из внешней сети.
  6. Система автоматически выполнит переход на страницу сервиса, который был выбран вами при регистрации для дальнейшей работы с ним

ВАЖНО ЗНАТЬ!

  1. При последующей работе с сервисами Личного кабинета из внутренней корпоративной сети (через корпоративный портал) авторизация пользователя будет выполняться автоматически при входе на портал.
  2. При дальнейшей работе с сервисами Личного кабинета из внешней сети интернет потребуется прохождение авторизации по номеру мобильного телефона и «пин-коду».
  3. В случае изменения номера мобильного телефона необходимо сообщить информацию в кадровую службу вашего предприятия для корректировки информации в системе, после чего на новый номер телефона будет выслан новый «пин-код».
  4. Изменить «пин-код» на другой удобный для вас 6-ти значный цифровой код можно в разделе «смена пароля» Личного кабинета, для этого вам потребуется исходный «пин-код», полученный при регистрации.
